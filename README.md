# PR Website Bootstrap

This is a professional starter website for a PR business.

## Files

- `index.html` — full one-page website structure
- `styles.css` — responsive styling
- `assets/placeholders/` — keep your future images here

## Quick customize checklist

1. Update business name (`ElevatePR`) in `index.html`.
2. Replace contact details in the Contact section.
3. Replace placeholder metrics in the Results section.
4. Fill out each case study card with real client outcomes.
5. Swap placeholder image blocks with `<img src="assets/placeholders/your-image.jpg" alt="...">`.
6. Connect the form to your preferred backend (Formspree, Web3Forms, custom API, etc.).

## Suggested assets to prepare later

- Hero image (wide, 1600px+)
- Founder/team portrait
- 3–6 case-study visuals
- Press logos or publication screenshots

## Local preview

Open `index.html` in your browser, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080/pr-website/`.
